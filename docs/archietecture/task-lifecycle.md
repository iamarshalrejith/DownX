## 1. Task Creation

Actor: Teacher or Parent

Steps:
1. Adult creates task with:
   - Title
   - Description
   - Difficulty level
2. Task saved in system
3. Task assigned to one or more students

---

## 2. AI Simplification

1. Task content sent to AI Simplifier
2. AI generates:
   - Simplified instructions
   - Step-by-step breakdown
3. AI output reviewed automatically
4. Final version stored per student

---

## 3. Task Consumption (Student)

1. Student logs in
2. Dashboard fetches assigned tasks
3. Student opens task
4. Instructions shown in child-friendly format
5. Student performs task

---

## 4. Task Completion

1. Student marks task as completed
2. Backend records:
   - Completion time
   - Completion status
3. Progress reflected in dashboards

---

## 5. Task Review

Actor: Teacher / Parent

- View completion history
- Identify delays or struggles
- Adjust future tasks accordingly
